# Personal Health Coach AI

This is a Streamlit-based AI application that monitors your health data (sleep & steps) and gives personalized recommendations.

## Features
- Enter sleep hours and steps walked
- Get health summary (average)
- Personalized recommendations
- Easy and interactive UI
